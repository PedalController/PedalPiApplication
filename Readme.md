## Local test

```bash
coverage3 run --source=architecture,controller,dao,model setup.py test
coverage3 report
coverage3 html
firefox htmlcov/index.html
```

## Using

```
jackd -P70 -p16 -t2000 -dalsa -dhw:Series -p128 -n3 -r44100 -s &
./mod-host
qjackctl &
```

killall -9 jackd
